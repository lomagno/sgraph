Create charts by retrieving data from Stata on-the-fly

Getting started
  1. Download SWire
  2. Install SWire on Stata (see SWire documentation)
  3. Start SWire in Stata in HTTP mode: swire start
  4. Open http://localhost:50000/test to check if the SWire HTTP server is running in HTTP mode on port 50000
  5. Use SGraph: click on Line chart or 3D scatter
